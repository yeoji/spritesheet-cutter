## Spritesheet Cutter

This project uses [Sprite Cow](https://github.com/jakearchibald/sprite-cow) as a base for sprite selection. It is extended with spritesheet cutting and export features.

### Development

```
yarn dev
```

Go to `http://localhost:1234`
